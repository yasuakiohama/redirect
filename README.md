#作成方法

    cd yourRepository/
    git checkout --orphan gh-pages
    git rm -rf . #これは別にやらなくてもいいかも
    echo "My GitHub Page" > index.html
    git add index.html
    git commit -a -m "First pages commit"
    git push origin gh-pages

    http://yourself-account.github.io/yourRepositoryアクセスするだけ。
